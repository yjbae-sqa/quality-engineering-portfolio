# Load Testing with JMeter

## Goal

Understand system behavior under concurrent user load.

## Test Flow

1. define API or endpoint target
2. configure virtual users
3. execute load test
4. observe response time and throughput

## Key Checks

- response latency
- request throughput
- error rate under load

## Example Scenarios

- concurrent API requests
- increasing user load
- basic system stress testing

## Tools

- Apache JMeter
